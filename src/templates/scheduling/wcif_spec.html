CubingUSA WCIF Extensions

Introduction

This page defines extensions to the WCIF specification used by the CubingUSA scheduling system.

While other applications may use data stored in these extensions, we reserve the right to modify the format of this extension in non-backwards-compatible ways.

org.cubingusa.ScheduleCompetition

The ScheduleCompetition extension is added to WCIF Competition entities. This extension represents a ScheduleCompetition entity in the App Engine datastore. The extension contains the following data:

  • datastoreId: (string) The ID of the entity stored in the App Engine datastore. This is the same as the WCA competition ID.
  • staffSignupDeadline: (YYYY-MM-DD) The last date that people may apply to be staff members and update their information.

org.cubingusa.Schedule

The Schedule extension is added to WCIF Schedule entities. This extension represents a Schedule entity in the App Engine datastore. The extension contains the following data:

  • datastoreId: (integer) The ID of the entity stored in the App Engine datastore.
  • creationTime: (ISO-8601 timestamp) The time that the schedule was created.
  • lastUpdateTime: (ISO-8601 timestamp) The time that the schedule was last updated.

org.cubingusa.ScheduleStage

The ScheduleStage extension is added to WCIF Room entities. This extension represents a ScheduleRoom entity in the App Engine datastore. The extension contains the following data:

  • datastoreId: (string) The ID of the entity stored in the App Engine datastore.
  • colorCss: (string) The CSS name of the stage's color.
  • colorHex: (string) The hex code of the stage's color.
  • numTimers: (integer) The number of timers used on this stage.

org.cubingusa.ScheduleTimeBlock

The ScheduleTimeBlock extension is added to WCIF Activity entities. These correspond to Activities which do not correspond to a single group. This extension represents a ScheduleTimeBlock entity in the App Engine datastore. The extension contains the following data:

  • datastoreId: (integer) The ID of the entity stored in the App Engine datastore.
  • staffOnly: (boolean) Whether the groups in this time block are limited to staff members only.
